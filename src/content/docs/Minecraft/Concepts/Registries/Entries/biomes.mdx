environment like the plants, mobs, structures, weather, and even ores.

## Biome Tags
Minecraft uses biome tags to classify and organize biomes. In some cases biome tags also influence gameplay functions.
For example, snow golems will melt in any biome in the `snow_golem_melts` tag, including biomes added to the tag by mods
and data packs.

The table below contains all vanilla biome tags for Minecraft 1.20.1. You can also find biome tags added by mod loaders
by selecting the respective tab at the top of the table.
