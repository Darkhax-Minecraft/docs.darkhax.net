This table includes every tip that is included with the base mod. The tip ID can be used to disable the tip by adding it
to the `ignoredTips` section of your config file.

:::tip[Disable All Built-in Tips]
If you are trying to disable all of the default tips you should add `tipsmod` to the `ignoredNamespaces` section of your
config file instead of writing out every tip ID in `ignoredtips`.
:::
